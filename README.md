# Coding in R
Slides and interactive tutorials for a PhD-level introductory course to coding in R.

- The `syllabus.html` and `schedule.html` contain the description of the course.
- The `slides/` folder contains the .qmd source files and the corresponding .html files with the slides for each class.
- The `tutorials/` folder contains a series of tutorials and exercises developed in [https://swirlstats.com/](swirl).
